This module will be used for financial reporting of revenue , cost , CM and CM%.

Budget: Budget is created at the start of year predicting the financial parameters for a project for a complete year.

Forecast: Forecast is the process of predicting the financial parameters for remaining months of the year. Example On 30Th June prediction will from July to December.

Actuals: Actual Financial data till the current month is called Actuals. Example Example on 30Th June actual figures from Jan to June will be actuals.

The main functions of Finance module are

1. Take Input from Forecast Excel and upload to Database.

2. Take Input from PCM Excel (Actuals) and upload to Database.

3. Create different views on Dashboard using forecast and actuals data for project . Different views include monthly , quarterly and yearly charts and graphs.
